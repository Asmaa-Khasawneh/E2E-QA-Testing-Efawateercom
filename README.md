# End-to-End QA Testing Project

This project demonstrates full QA testing for a web application similar to E-Fawateercom.

## Overview

Performed comprehensive testing covering multiple testing types to ensure system quality, functionality, and performance.

## Testing Types

* API Testing using Postman
* Automation Testing using Selenium WebDriver (C#)
* Performance Testing using JMeter

## Project Structure

* API Testing (Test cases + Postman collection)
* Automation Testing (Selenium scripts + test results)
* Performance Testing (JMeter scripts + reports)

## Key Features Tested

* User authentication (Login)
* Core system functionalities
* Data validation
* System performance under load

## Tools Used

* Postman
* Selenium WebDriver (C#)
* JMeter

## Outcome

This project demonstrates my ability to perform end-to-end testing, identify issues, and ensure high-quality software delivery using professional QA tools.
